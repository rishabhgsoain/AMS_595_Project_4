## How to Run Each Script

Each Python file is self-contained and can be executed directly from the terminal.

---

### **1. Mandelbrot Set Visualization**

Generates the Mandelbrot fractal image over the region `[-2, 1] × [-1.5, 1.5]`.

```bash
python mandelbrot.py
```

**Output:**

* Displays a grayscale fractal image.
* Saves the figure automatically as `mandelbrot.png`.

**Interpretation:**

* **White points:** Bounded values (inside the Mandelbrot set)
* **Black points:** Divergent values (outside the set)

---

### **2. Markov Chain Simulation**

Creates a random 5-state Markov chain, evolves it for 50 transitions, and compares the resulting distribution with its stationary distribution.

```bash
python markov_chain.py
```

**Output:**

* Prints the transition matrix, initial and final distributions, and their component-wise differences.
* Verifies if the final distribution converges to the stationary distribution within a tolerance of `1e-5`.

**Example Output:**

```
Transition Matrix P:
[[0.12 0.23 0.19 0.25 0.21] ...]
Match within 1e-5? -> True
```

---

### **3. Taylor Series Approximation**

Approximates the function
[
f(x) = x\sin^2(x) + \cos(x)
]
on the interval `[-10, 10]` using truncated Taylor polynomials of various degrees.

```bash
python taylor.py
```

**Outputs:**

* Displays a plot comparing the true function and its Taylor approximation.
* Saves a CSV file named `taylor_values.csv` containing:

  * `degree` — truncation level
  * `totalError` — sum of absolute errors
  * `timeSeconds` — computation time

**Sample Output:**

```
   degree     totalError  timeSeconds
0       50  1.98234e+03     0.1224
1       60  4.56789e+02     0.1432
...
Wrote: taylor_values.csv
```
